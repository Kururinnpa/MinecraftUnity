# MinecraftUnity
使用Unity复现的Minecraft游戏

## 主要内容
### 2025.2.17
* 使用柏林噪声随机生成地图（目前仅支持平原地形）
* 按照block-chunk-world结构维护方块，以chunk为单位构建world
* 将chunk中所有方块的表面合并成一个，提升渲染性能
* 实现背包系统，包括大背包、背包内快捷栏与背包外快捷栏
* 方块的建造与破坏
* 区分生存与创造两种模式，目前生存模式暂未加入生命值系统与怪物
